Each byte is framed like classic async serial: a start bit (0) pulls the idle-high line low, then 8 data bits, least-significant first, then a stop bit (1) returns the line to idle. "Time on wire" assumes 115200 baud, 10 bits/byte, and no compression — a rough real-world stand-in so the three formats can be compared on equal footing. Avro relies on the schema instead of delimiters; JSON and XML pay for self-description in extra bytes (and XML's closing tags cost more than JSON's punctuation).
